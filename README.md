Employee Performance Analysis using SQL

## Problem Statement
Analyze employee salary and performance data to extract meaningful business insights.

## Dataset
- Employee Table: Salary, Job Role, Join Date
- Performance Table: Rating, Department, Experience

## Tools Used
- SQL (MySQL)

## Key Insights
- Top 5 highest-paid employees
- Employees earning above average salary
- Salary ranking using window functions
- Department-wise performance analysis

## Project Structure
- 01_database_setup.sql → Database setup
- 02_basic_queries.sql → Basic SQL queries
- 03_advanced_analysis.sql → Business insights
